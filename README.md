
## Description

[Nest](https://github.com/nestjs/nest) AdminV3.

## Installation

```bash
$ yarn
```

## Running the app docker

```bash
$ docker-compose build
$ docker-compose up
```

## Running the app locally

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
